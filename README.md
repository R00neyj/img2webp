# img2webp

이미지(JPG, PNG 등)를 WebP로 일괄 변환해주는 간단한 도구입니다

## 특징

- **빠름**: sharp 라이브러리를 사용하여 변환 속도가 매우 빠릅니다
- **편함**: run.bat만 실행하면 패키지 설치부터 변환까지 자동으로 진행합니다
- **품질 조절 가능**: 90%(상), 80%(중), 70%(하) 품질 중 선택이 가능합니다

## 사전 준비

- [Node.js](https://nodejs.org/) v18 이상이 설치되어 있어야 합니다

## 사용 방법

1. 프로젝트 폴더를 적당한 위치에 둡니다
2. input 폴더에 변환할 이미지를 넣습니다 (폴더가 없으면 run.bat 실행 시 자동 생성됩니다)
3. run.bat 파일을 실행합니다
4. 원하는 품질 번호(1~3)를 입력하고 엔터를 누릅니다
5. output 폴더에 생성된 .webp 파일을 확인합니다

## 폴더 구조

- input/: 원본 이미지 경로 (.jpg, .png, .gif 등)
- output/: 변환된 .webp 파일 저장 경로
- run.bat: 윈도우용 원클릭 실행 스크립트
- src/: 핵심 로직 및 설정 파일들이 포함된 폴더

## 지원 확장자

- .jpg, .jpeg, .png, .tiff, .gif, .bmp
