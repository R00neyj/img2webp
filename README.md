# img2webp

이미지(JPG, PNG, TIFF, GIF, BMP)를 고효율 WebP 포맷으로 일괄 변환하는 Node.js 기반의 강력한 도구입니다.

## 주요 특징

- **고성능 병렬 처리**: `sharp` 라이브러리와 `Promise.all` 기반의 배치(Batch) 처리 방식을 도입하여 대량의 이미지도 매우 빠르게 변환합니다.
- **실시간 진행 상태**: 처리 중인 파일 수와 성공/실패 여부를 실시간으로 출력하여 작업 현황을 쉽게 파악할 수 있습니다.
- **유연한 품질 조절**: 프리셋(최상/상/중/하) 제공은 물론, 사용자가 직접 1~100 사이의 품질 수치를 지정할 수 있습니다.
- **원클릭 실행**: `run.bat` 파일을 통해 의존성 설치부터 실행까지 한 번에 처리할 수 있습니다.

## 사전 준비

- [Node.js](https://nodejs.org/) v18 이상이 설치되어 있어야 합니다.

## 사용 방법

1. 변환할 이미지들을 `input` 폴더에 넣습니다. (폴더가 없으면 첫 실행 시 자동 생성됩니다.)
2. `run.bat` 파일을 실행합니다. (또는 `node src/convert.js` 명령어를 사용합니다.)
3. 메뉴에서 원하는 품질 번호(1~5)를 입력하고 엔터를 누릅니다.
4. 모든 변환이 완료되면 요약 보고서가 표시됩니다. 엔터를 누르면 다시 메인 메뉴로 돌아가 다른 품질 설정을 반복할 수 있습니다.
5. `output` 폴더에서 생성된 `.webp` 파일들을 확인합니다.

## 폴더 구조

```text
img2webp/
├── input/          # 변환할 원본 이미지 저장 폴더 (자동 생성)
├── output/         # 변환된 .webp 파일 저장 폴더 (자동 생성)
├── src/            # 핵심 소스 코드 및 패키지 설정
│   ├── convert.js  # 메인 변환 로직 및 CLI 인터페이스
│   ├── package.json
│   └── node_modules/
├── run.bat         # Windows용 자동 설치 및 실행 스크립트
└── README.md       # 프로젝트 개요 및 사용법
```
